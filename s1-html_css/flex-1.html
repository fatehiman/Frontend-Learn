  1. display:flex باید به کانتینر داده بشه
  2. display:flex کل چایلدها رو از حالت بلاک خارج میکنه
  3. flex قابلیت shrink داره یعنی فشرده میکنه که جا بشه و از والدش نزنه بیرون
  4. اگر بخوایم این نسبت shrink درمورد یک المنت متفاوت باشه از flex-shrink استفاده میکنیم
  5. flex-direction مشخص میکنه محور اصلی کدوم باشه. پیشفرض row هست
  6. justify-content برای چیدمان افقی (محور اصلی) المان هاست
  7. align-content برای چیدمان عمودی (محور فرعی) المان هاست
